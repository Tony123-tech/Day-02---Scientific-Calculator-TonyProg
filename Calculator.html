About This Calculator

This is a modern, responsive scientific calculator built with HTML, Tailwind CSS, and JavaScript. It supports basic arithmetic, advanced scientific functions, and three stylish themes: Light, Dark, and Cyberpunk!

  • Basic arithmetic: +, -, ×, ÷, decimals
  • Parentheses, exponentiation (xʸ), square root (√)
  • Trigonometric functions: sin, cos, tan
  • Logarithmic and exponential: log, ln, exp
  • Constants: π, e
  • Adaptive design: works great on desktop and mobile
  • Switch themes between Light, Dark, and Cyberpunk

Note: All calculations use radians for trigonometric functions.
